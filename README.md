# mi_aplicacion_gui
# Aplicación GUI para Registro de Personas

## Descripción

Este proyecto consiste en el desarrollo de una aplicación con interfaz gráfica de usuario (GUI) que permite registrar información básica de personas. La aplicación fue desarrollada en Python utilizando la librería Tkinter y aplicando principios de Programación Orientada a Objetos (POO) para mantener una estructura organizada y modular del código.

La aplicación permite al usuario ingresar datos como el nombre y la cédula de una persona a través de campos de texto. Posteriormente, la información ingresada se muestra en una lista dentro de la interfaz gráfica, permitiendo visualizar los datos agregados de forma sencilla.

## Funcionalidades

* Ingreso de nombre y número de cédula mediante campos de texto.
* Botón **Agregar** para registrar los datos ingresados.
* Visualización de los registros dentro de una lista en la interfaz.
* Botón **Limpiar** para eliminar los datos registrados.
* Manejo de eventos generados por la interacción del usuario con la interfaz.

## Tecnologías utilizadas

* Python
* Tkinter
* Programación Orientada a Objetos (POO)

## Requisitos

Para ejecutar el proyecto es necesario tener instalado:

* Python 3.x

Tkinter ya viene incluido en la mayoría de las instalaciones de Python.

## Ejecución del proyecto

Para ejecutar la aplicación, abre una terminal en la carpeta del proyecto y ejecuta:

```
python main.py
```
